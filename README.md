# TextProcessor
## Описание программы

Программа принимает на вход файл на диске. Обработка файла заключается в
следующем:
a. удаление слов длиной менее какого-либо числа символов;
b. удаление знаков препинания.
Результат сохраняется в файл.

## Реализованные возможности

Входной файл, выходной файл, длину слов и
необходимость удаления знаков препинания должен задает пользователь c помощью интерфейса.
По завершению обработки всплывает окно с сообщением об этом.
Во время обработки пользователь всё ещё может взаимодействовать с интерфейсом, программа не "зависает".

## Руководство пользователя

Для обработки текстовых документов, вам необходимо открыть те файлы, которые хотите отредактировать с помощью кнопки "Выбрать файл(ы)"
Вы можете выбрать как один документ, так и несколько.
С помощью флажков в правой стороне окна, вы можете выбрать, как именно хотите обработать файл - удалить знаки пунктуации
и/или удалить слова длиной менее определенного количества символов (вы можете сделать это в любой момент, даже до выбора файлов).
После этого, нажмите на кнопку "Сохранить файл(ы)" для сохранения. Для каждого файла запрашивается место сохранения.
Дождитесь сообщения об окончании обработки файлов. Готово.
